# github-snake-generator

Snake animado que se va comiendo los recuadros de contribuciones de GitHub, creciendo sobre la marcha y con version clara, oscura y GIF.

## Output files

- `github-snake.svg`: version clara para README o perfil.
- `github-snake-dark.svg`: variante adaptada a modo oscuro.
- `github-snake.gif`: animacion con fondo oscuro y colores mas llamativos.
